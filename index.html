Checklist qualité en magasin
1. Liste d'ingrédients : max 5, noms reconnaissables
2. Origine : "Origine France" (pas juste "Élaboré en")
3. Oeufs : code 0 (bio) ou 1 (plein air) sur la coquille
4. Dirty Dozen = BIO obligatoire : pommes, fraises, épinards, raisin, poivrons, tomates, brocolis
5. Emballage : verre > inox > plastique. Conserve acide (tomates) = éviter
6. Pas de : huile de palme, E250/252 (nitrites), E621 (glutamate), sucre dans les 3 premiers ingrédients
Meal prep dimanche
Préparer Lun-Jeu en batch (~90 min). Vendredi = poisson froid, pas de cuisson.
Conservation : viande cuite 4 jours max au frigo, boîtes en verre.
Lavage légumes : trempage 10-15 min eau froide + bicarbonate de soude, rincer, peler si non-bio.
